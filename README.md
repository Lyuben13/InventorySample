# 🏪 Система за управление на склада

[![.NET](https://img.shields.io/badge/.NET-8.0-purple.svg)](https://dotnet.microsoft.com/)
[![C#](https://img.shields.io/badge/C%23-12.0-blue.svg)](https://docs.microsoft.com/en-us/dotnet/csharp/)
[![Tests](https://img.shields.io/badge/Tests-12%20Passed-brightgreen.svg)](InventoryApp.Tests)

> Конзолно приложение за управление на складови наличности.
> Разработено с C# и .NET 8.

## Основни функционалности

- ✅ Добавяне на нов продукт
- ✅ Редактиране на съществуващ продукт
- ✅ Изтриване на продукт по ID
- ✅ Преглед на всички продукти
- ✅ Търсене по име, ID и доставчик
- ✅ Филтриране по наличност и ценови диапазон
- ✅ Сортиране по различни критерии
- ✅ Генериране на складов отчет

## Структура на проекта

```
InventoryApp/
├── Models/
│   └── Product.cs
├── Services/
│   ├── IInventoryService.cs
│   └── InventoryService.cs
├── Reports/
│   └── InventoryReportService.cs
├── Extensions/
│   └── ConsoleMenu.cs
└── Program.cs
```

Проектът е организиран така, че бизнес логиката, моделите, отчетите и помощните функции да са разделени в отделни компоненти.

## Ограничения на текущата версия

- Данните се съхраняват в паметта по време на изпълнение
- Няма връзка с база данни
- Има unit тестове за основните функционалности
- Няма графичен интерфейс
